* [OpenCV >= 3.2] (https://opencv.org/)
* [QT >= 5] (https://www.qt.io/)
* [OpenGL]
* [GLM]
* [CUDA >= 8.0] (https://developer.nvidia.com/cuda-downloads)
* [FLANN] (http://www.cs.ubc.ca/research/flann)
* [Eigen] (http://eigen.tuxfamily.org)
